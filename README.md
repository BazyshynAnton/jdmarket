## Features

-Log in. <br/>
-Edit your profile. <br/>
-If you're not logged in, register. <br/>
-Add cars from the catalog to your "Favorites." <br/>
-Filter the car catalog. <br/>
-Search for cars in the search field.

## Technologies

-React.  
-React-Router-DOM.  
-Redux Tool Kit for state management.  
-SCSS for styling.  
-vite.  
and other.
